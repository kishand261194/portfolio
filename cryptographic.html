Cryptographic Tools

Source code GitHub

Programming Language used: Python 3
Imports/Libraries used: pycryptodome, matplotlib

Commands to install dependences:
- sudo pip install pycryptodome
- sudo pip install pycryptodomex

Additional(Graph can be commented out): matplotlib
- python -mpip install matplotlib

Reference: pycryptodome

The program will use different cryptographic functions and measure the speed of each. For that purpose, it will create a small file of size 1KB and a large file of size 1MB. These are randomly generated data. The program is to implement the following functionalities:
(a) Creates a 128-bit AES key, encrypt and decrypt each of the two files using AES in the CBC mode.
(b) Creates a 128-bit AES key, encrypt and decrypt each of the two files using AES in the CTR mode.
(c) Repeats part (b) with a 256-bit key. two files using AES in the CTR mode.
(d) Computes a hash of each of the files using hash functions SHA-256, SHA-512, and SHA3-256.
(e) Creates a 2048-bit RSA key, encrypt and decrypt the files above with PKCS #1 v2 padding (it is also be called OAEP).
(f) Repeats part (e) with a 3072-bit key.
(g) Creates a 2048-bit DSA key, sign the two files and verify the corresponding signatures. Creating a key takes two parameters, uses 224 bits for the exponent sizes. and hash function algorithm uses SHA-256.
(h) Repeats part (g) with a 3072-bit DSA key.
The above github link has each implemention in python, also has the report attached , in which each result is compared with one other.
